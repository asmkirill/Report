This is a description of a Django project.

"Minutes" folder: This folder contains tech files that help the project work.
"Main" folder: This folder has a subfolder called "templates" with two files: index.html: This file has a form where the user enters a document number and clicks a button. When the button is clicked, the corresponding document is retrieved from the MySQL database. If the document number is not found, an error message is displayed. Additionally, there is a button that takes the user to another page.
layout.html: This is a base template used for other pages in the project. "protocol" folder: This folder has a subfolder called "templates" with one file: create_protocol.html: This file has a form that displays seven fields.
When the page is loaded, the document number is displayed at the top of the form.

There are two buttons at the bottom of the form:

"Add": When this button is clicked, five empty fields are added to the form.
"Save": When this button is clicked, the form data is saved without reloading the page.
